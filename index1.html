Important FAQ

What is the Document Object Model (DOM)?

DOM works as a broker between HTML and JAVASCRIPT

How do you an Element form the DOM?

we can use properties to select element , to choose a particular element use getElementById() , or to get several elements use getElementByClassName() or to get common tag names in the whole HTML use getElementByTagName()

What event delegation in the contest of DOM, and why is it useful?

Event delegation helps to access child elements of a parent tag while implementing event handler, rather than giving each element a common event handler we can access through their parent

How do you manipulate an element's attributes and styles using the DOM

after getting an element using getElementByClassName() or getElementById() of getElementByTagName() we can add property (.style) to give the elements different styles. As for attributes we can use (.setAttribute('name', 'value')) to set any attribute to the elements
